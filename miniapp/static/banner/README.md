# 轮播图图片说明

## 需要的图片文件

请在 `2/static/` 目录下创建 `banner` 文件夹，并添加以下5张轮播图：

### 图片要求：
- 建议尺寸：800x400 像素
- 格式：JPG 或 PNG
- 文件名：
  - `banner1.jpg` - 警惕电信诈骗
  - `banner2.jpg` - 刷单返利都是骗局
  - `banner3.jpg` - 保护个人信息
  - `banner4.jpg` - 遇到诈骗及时报警
  - `banner5.jpg` - 学习防诈骗知识

### 目录结构：
```
2/
└── static/
    └── banner/
        ├── banner1.jpg
        ├── banner2.jpg
        ├── banner3.jpg
        ├── banner4.jpg
        └── banner5.jpg
```

### 轮播图配置：
轮播图数据已在首页中配置，包括：
- 标题
- 图片路径
- 跳转类型
- 关联内容ID

### 点击跳转功能：
- 知识类轮播图：点击跳转到对应的知识详情页
- 举报类轮播图：点击跳转到举报页面

### 注意事项：
1. 确保图片文件名与代码中的配置一致
2. 图片大小建议控制在500KB以内，提高加载速度
3. 可以使用设计软件制作精美的防诈骗主题图片
4. 图片内容应与标题相符，突出防诈骗主题
